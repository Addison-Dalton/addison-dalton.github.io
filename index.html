Tea Time

This is an android app that allows you to time how long to brew tea. You can create your own tea profiles with how long the tea is to be brewed. Next time you want to brew that tea, simply select it from the list and start the timer.

RSA Encryption Demo

This program was written in C# for my capstone mathematics project. The program takes a message, encrypts it, shows the encrypted message, and also allows for decryption of the message. Other data show are variables and equations used within the RSA encryption algorithm.

Solar System Simulation

This program was written in C++ with AppGamingKit. It was created for my Model and Simulation capstone project. The program simulates a small solar system. Users have the ability to move celestial objects, change their mass, and change their velocity. Users can also add new celetial objects into the simulation.

JavaScript Calculator

Simple javascript-based calculator that allows the user to do operations such as: multiplication, division, subtraction, and addition. Included is a clear all button, a delete button, a +/- change button, and a percent button. The calculator can take input from the on-screen buttons or keyboard inputs.

Tic-Tac-Toe

This is a web-based tic-tac-toe game. Users has the ability to choose between 'X' and 'O'. A user can play multiple games against the computer, with score being kept with each game. There is a rest button to reset the score.
